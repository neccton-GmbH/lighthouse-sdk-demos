# Lighthouse Demo App (iOS SwiftUI)

Reference iOS demo for the Lighthouse SDK.

## What it shows

- collecting tenant key, player ID, and environment
- requesting a refresh token from the backend
- initializing the Lighthouse SDK session
- loading, reading, archiving, and restoring messages

## Package

This demo is configured to use the public `lighthouse-sdk-ios-dist` package.

## Open in Xcode

Open `LighthouseDemoApp.xcodeproj` and run the `LighthouseDemoApp` scheme.
