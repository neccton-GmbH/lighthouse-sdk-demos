# Lighthouse Demo App (Web Vite)

Reference web demo for the Lighthouse SDK.

## What it shows

- collecting API key, customer ID, and environment
- requesting a refresh token from the backend
- initializing the Lighthouse SDK session
- loading unread, active, archived, and all messages
- reading, archiving, restoring, and engaging message actions

## Dependency

This demo is configured to use the published `@neccton-gmbh/lighthouse-web` package.

## Run locally

```bash
npm install
npm run dev
```
