# Lighthouse Demo App (Android Compose)

Reference Android demo for the Lighthouse SDK.

## What it shows

- collecting refresh token, environment, base URL, language, and country
- initializing the Lighthouse SDK session
- loading active, unread, and archived messages
- archiving and restoring sample messages from the loaded result set

## Dependency

This demo is configured to use the public `lighthouse-sdk-android-dist` artifact.

## Open in Android Studio

Open `android/compose/LighthouseDemoApp` as a Gradle project.
